# IonicMovies
IonicMovies is an App built with Ionic and Angular. It was my first project when relearning Ionic with Angular after some time away from this technologies.

# Tech Stack
- Ionic
- Angular
- TypeScript
- TMDB API

## Faetures
 - Browse All Movies: Explore a wide range of movies available on the platform.
 - Search Movies: Easily search for specific movies using a search function.
 - Movie Details Screen: See movie details such as the full poster, genres, release date and budget.
 - Dark Mode: Seamlessly switch between light and dark themes for a comfortable and visually appealing user experience in any environment.
 - Modern UI/UX: A sleek and user-friendly interface designed for a great experience.
 - Responsiveness: Fully responsive design that works seamlessly across devices.

## Experience IonicMovies by yourself
 1. Clone this repo
 2. Run `npm install` to install all the packages need to run this project
 3. Run `ionic serve` to start the server and see the app in your browser
